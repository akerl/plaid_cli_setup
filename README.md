[Plaid](https://plaid.com/) API key workflow

## Usage

Run the web service

```
# Fill in your API credentials
export PLAID_ENV=development
export PLAID_CLIENT_ID=
export PLAID_PUBLIC_KEY=
export PLAID_SECRET=
ruby app.rb
```

Then open your browser and navigate to the app.
